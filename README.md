# Cold Leyu Journal

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为静态页面集合的存储与展示空间。

## 项目简介

`cold-leyu-journal` 旨在收集、整理并托管一系列独立的 HTML 文件。这些页面可能包含笔记、文档、工具界面或实验性内容，彼此之间无强关联，各自独立运行。

## 目录说明

```
.
├── index.html          # 仓库主页（可选，用于导航或索引）
├── pages/              # 存放主要 HTML 页面文件
│   ├── page1.html
│   └── page2.html
├── assets/             # 页面所需的静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：每个 HTML 文件视为一个独立页面，可直接通过浏览器打开或通过 GitHub Pages 等服务访问。
- `assets/`：存放页面引用的样式、脚本或媒体文件，按需组织。

## 页面归档说明

- 每个 HTML 页面均为自包含或引用仓库内资源，不依赖外部第三方服务。
- 页面内容可能包含个人笔记、工具、演示或实验性项目，不保证持续更新或兼容性。
- 归档内容仅供学习、参考或存档用途，请勿用于商业或违规场景。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修正已有内容。
- 欢迎通过 Issue 或 Pull Request 提出建议或修正，但请注意本仓库不针对任何特定主题或项目。
- 维护者保留对页面内容进行修改、移除或归档的权利。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件，或通过本地服务器（如 `python -m http.server`）运行。
3. 若启用 GitHub Pages，请将根目录或 `docs/` 文件夹设置为发布源，即可通过对应 URL 访问。
